## NeoDash 
NeoDash is a dashboard builder for Neo4j, letting you build a graph dashboard in minutes.

**This project is no longer maintained, use at your own risk.** 


1. For the best experience, [upgrade](https://console-preview.neo4j.io/tools/dashboards) to the new Neo4j Dashboards tool.
2. If you'd like to keep using NeoDash, you can fork this repository and [run NeoDash yourself](https://github.com/neo4j-contrib/neodash/blob/master/about.md).
